# 🐳 Whally Copilot

Gestionnaire de portefeuilles crypto multi-exchange pour traders caribéens.

## Wallets supportés
- Binance (BTC, ETH, BNB, USDT)
- Phantom / Solflare (SOL, IKB token)
- Trust Wallet
- MEXC
- Base (ETH, USDC)

## Stack
React 18 + TypeScript + Vite + TailwindCSS + Recharts

## Démarrage
```bash
npm install && cp .env.example .env && npm run dev
```
